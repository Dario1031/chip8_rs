# chip8_rs

A simple CHIP-8 interpreter written in Rust. This project was built as a learning exercise to better understand emulator architecture and the Rust programming language.

## Features

* CHIP-8 CPU implementation
* Desktop frontend using `minifb`
* WebAssembly support for running in the browser
* Platform-independent emulator core

## Requirements

* Rust (latest stable)
* Cargo

For the WebAssembly build:

* `wasm-pack`
* `wasm32-unknown-unknown` Rust target

## Desktop

Run the desktop emulator with:

```bash
cargo run -- <scale> <delay_ms> <rom>
```

Example:

```bash
cargo run -- 10 2 Tetris.ch8
```

Arguments:

* `scale` – Window scale factor (e.g. `10`)
* `delay_ms` – Delay between CPU cycles in milliseconds
* `rom` – Path to the CHIP-8 ROM

## WebAssembly

Install the required tools:

```bash
rustup target add wasm32-unknown-unknown
cargo install wasm-pack
```

Build the WebAssembly package:

```bash
wasm-pack build --target web
```

This generates a `pkg/` directory containing the compiled WebAssembly module and JavaScript bindings.

> **Note:** The `pkg/` directory is generated and is intentionally not committed to the repository. Anyone cloning the project should generate it locally using the command above.

## Project Structure

```text
src/
├── chip8.rs      # Emulator core
├── platform.rs   # Desktop (minifb) frontend
├── main.rs       # Desktop entry point
└── lib.rs        # WebAssembly bindings

web/              # Browser frontend
pkg/              # Generated by wasm-pack (ignored by Git)
```
